**技术_软件开发_BinauralStudio双耳节拍工作室**

**来源**：网络

**概念**：<mark>Binaural Studio</mark> 是一个本地优先的浏览器内双耳节拍工作室——结合十声部双耳合成器、本地音乐播放器、独立语音播放器和实时录音器。可在浏览器内分析音乐调性/BPM/响度，自动调谐生成层匹配每首曲目，在演奏过程中切换状态预设，并录制完整结果为单个音频文件。无后端、无账号、无构建步骤、不上传音频。

**子概念**：
- <mark>双耳合成器</mark>：7种状态预设、10个独立混音谐波对、1-40Hz拍频、可调节载波/正弦三角振荡器、手动输出控制
- <mark>音乐分析</mark>（浏览器本地）：整轨色度图谱估算调性（含置信度）、ONSET包络估算节奏/BPM/节拍相位、响度和静音段落映射；低置信度保留原载波
- <mark>自适应层</mark>：慢速双耳自动调平、脉冲纹理、滤波气垫、呼吸调制、由当前曲目塑造的空间运动
- <mark>双Deck架构</mark>：Deck 01音乐（EQ/调性/节奏分析驱动自动生成层）；Deck 02语音（独立音量/混响/位置/静音间隔，不影响音乐分析）
- <mark>录音与隐私</mark>：实时录制完整主混音为M4A/WEBM/OGG；支持File System Access API直接写入磁盘（否则存内存后下载）；所有音频数据和录音仅存本地设备，无分析/服务器组件
- <mark>技术栈</mark>：纯HTML+CSS+JS，零依赖，单页面应用；Web Audio API 驱动；python3 http.server即可本地运行；MIT许可证
